# Windows မှာ Visual Studio Code (VSCode) Install လုပ်နည်း

Visual Studio Code (VSCode) သည် Microsoft မှတည်ဆောက်ထားသော lightweight နဲ့ powerful text editor တစ်ခုဖြစ်ပြီး Web Development, Python, JavaScript, Cybersecurity စတဲ့လုပ်ငန်းများတွင်အထူးအသုံးပြုသည်။

ဒီ guide မှာ Windows 10/11 တွင် VSCode ကို install လုပ်နည်းကိုအဆင့်လိုက်ဖော်ပြပေးပါမယ်။

---

## 🔧 လိုအပ်ချက်များ

- Windows 10/11 Operating System
- Internet Connection

---

## 🚀 Installation Steps

### ၁။ VSCode Website သွားပါ

👉 [https://code.visualstudio.com/](https://code.visualstudio.com/)

<img src="images/1.png" alt="Vscode Download" width="600"/>

---

### ၂။ `Download for Windows` ကိုနှိပ်ပါ

<img src="images/2.png" alt="Vscode download" width="600"/>

Download file name: `VSCodeUserSetup-x64-*.exe`

---

### ၃။ Download ပြီးရင် Installer ကို run လုပ်ပါ

VSCode Installer ကို double-click နှိပ်ပြီး install လုပ်ပါ။

<img src="images/3.png" alt="Vscode run" width="600"/>

---

### ၄။ License Agreement ကို Accept လုပ်ပါ

✔️ “I accept the agreement” ကိုရွေးပြီး `Next` နှိပ်ပါ။

<img src="images/4.png" alt="vscode" width="600"/>

---

### ၅။ Install Location ရွေးပါ

Default location ကိုထားခဲ့လို့ရပါတယ်။ `Next` နှိပ်ပါ။

<img src="images/5.png" alt="Vscode" width="600"/>

---

### ၆။ Additional Tasks ရွေးပါ

✔️ တချို့သော options များကို check လုပ်ပေးသင့်ပါတယ်။

- ☑️ Add to PATH
- ☑️ Register Code as editor for supported file types
- ☑️ Add “Open with Code” action to Windows Explorer

<img src="images/6.png" alt="vscode" width="600"/>

---

### ၇။ Install လုပ်ပါ

`Install` ကိုနှိပ်ပြီး VSCode ကို install လုပ်ပါ။

<img src="images/7.png" alt="vscode" width="600"/>

---

### ၈။ Finish နဲ့ Launch

Finish ကိုနှိပ်ပြီး VSCode ကိုဖွင့်ပါ။

<img src="images/8.png" alt="vscode" width="600"/>

---

## ✅ Install ပြီးသွားတဲ့နောက် VSCode Interface

<img src="images/9.png" alt="vscode" width="600"/>

## 🎯 အသုံးဝင်သော Extensions များ

| Extension Name | Description |
|----------------|-------------|
| Python         | Python Development |
| Prettier       | Code Formatter |
| Live Server    | Launch local server |
| GitLens        | Git integration |
| ESLint         | JS/TS Syntax Checking |

---

## 📝 VSCode ကို Command Line မှဖွင့်ရန်

VSCode install လုပ်ပြီးသားဖြစ်ပါက Command Prompt မှာ `code .` လို့ရေးပြီး current folder ကို VSCode ထဲမှာဖွင့်နိုင်ပါတယ်။
